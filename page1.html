I: adding java script.
i: he HTML script tag is used to define a client-side script (JavaScript).

ii:the script element either contains script statements, or it points to an external script file through the src attribute.

iii: Common uses for JavaScript are image manipulation, form validation, and dynamic changes of content.

iv:To select an HTML element, JavaScript most often uses the document.getElementById() method.

II: tables:


i: Cell padding specifies the space between the cell content and its borders.If you do not specify a padding, the table cells will be displayed without padding.

ii: By default, table headings are bold and centered.To left-align the table headings, use the CSS text-align property.

III: symbols


i: Many mathematical, technical, and currency symbols, are not present on a normal keyboard. To add such symbols to an HTML page, you can use the entity name or the entity number (a decimal or a hexadecimal reference) for the symbol.
